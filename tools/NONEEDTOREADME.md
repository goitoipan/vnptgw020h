Nhập đường dẫn tới file .cfg khi được hỏi
- 0 để giải mã file và chỉnh sửa
- 1 để mã hoá file và up lên modem ( bắt buộc )
Lưu ý:
- khi giải mã, file sẽ có thêm đuôi .dec có thể mở bằng notepad hoặc bất cứ trình soạn văn bản nào, duôi này không có ý nghĩa với chương trình, chủ yếu tránh trùng tên
- tương tự thế, file khi mã hoá lại sẽ có đuôi .enc, phải sửa lại thành .cfg ( bỏ cả đuôi .dec và .enc ) lưu ý vẫn giữ tên là romfile thì mới upload lên modem được, chú ý tránh nhầm với file cũ là ok
